Password Generator

The Password Manager project is a C++ application designed to securely manage passwords for different websites. It includes the following key functionalities: Add Password: Stores a password for a specified website. Remove Password: Deletes the stored password for a given website. Retrieve Password: Fetches the password for a specified website. List All Passwords: Displays all stored site-password pairs. Generate Random Password: Creates a strong, random password of a specified length using a mix of characters, including numbers, uppercase and lowercase letters, and special symbols. The implementation uses standard C++ libraries for input/output operations, random number generation, and data storage in an unordered map. The code is structured with a class (PasswordManager) that encapsulates all the functionalities, ensuring a clean and modular design.
